Acerca

Este proyecto muestra un ejemplo de sitio multipágina desarrollado con Vite y LESS. Incluye diseño responsive, componentes reutilizables y estructura modular orientada a buenas prácticas de desarrollo web.

El objetivo es demostrar la integración de un preprocesador CSS dentro de un entorno moderno de construcción con Vite.
